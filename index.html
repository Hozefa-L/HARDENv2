Public Research Artifact

HARDEN-V2

HARDEN-V2 is a standalone public research artifact for rerunning the retained Phase 7 balanced smart-contract vulnerability benchmark from the included public inputs.

This public branch is organized for a reviewer: it keeps the active code path, the retained split/feature/graph inputs, and one clear run command instead of a broader private research workspace.

What this repo shows

The retained Phase 7 inputs, the code path they exercise, and the single public rerun command.

What this repo avoids

Paper assets, local workspace clutter, and generated outputs that a reviewer should produce locally.

Who this is for

Reviewers and researchers who need a human-readable guide to the public Phase 7 artifact.

Architecture pipeline

The public repo preserves the active path from benchmark construction to experiment analysis. Each stage below maps directly to code that remains in the repository.

1

Curation

Curate the CGT + DAppSCAN runtime-bytecode benchmark and harmonize labels.

2

Preprocessing

Build heterogeneous control-flow and data-flow graphs directly from runtime bytecode.

3

Features

Extract enriched opcode features, including TF-IDF and expert vulnerability patterns.

4

Models

Train OpcodeGT alongside retained classical and neural baseline models.

5

Evaluation

Aggregate run outputs and summarize publication-facing metrics with the retained analysis script.

Repository map

The public tree is intentionally small. These are the main paths to start with first.

src/

Phase 7 rerun path

The public branch keeps one reviewer-facing run path: install the dependencies, execute Phase 7, and inspect the generated outputs.

1. Install Python dependencies

pip install -r requirements.txt

2. Run the retained public experiment

python -m src.training.run_experiments --config configs/phase7_balanced.yaml

3. Review generated outputs

checkpoints/phase7_balanced/
reports/phase7_balanced/

4. Optional aggregate analysis

python scripts/level4_analysis.py --manifest reports/phase7_balanced/phase7_run_manifest.json
